# Seeders Documentation

## الوصف

يحتوي هذا المجلد على Seeders لإنشاء بيانات تجريبية للنظام.

## Seeders المتاحة

### 1. PositionSeeder
ينشئ 5 وظائف مختلفة:
- مدير عام
- مدير الموارد البشرية
- مدير المبيعات
- مدير تقنية المعلومات
- مدير المالية

### 2. ApplicantSeeder
ينشئ 11 متقدم مع:
- بيانات شخصية كاملة
- خبرات ومهارات واقعية
- توزيع على الوظائف المختلفة
- حساب النقاط التلقائية
- تقييمات (4-5 نجوم)

### 3. DatabaseSeeder
Seeder الرئيسي الذي يستدعي جميع Seeders الأخرى.

## الاستخدام

### تشغيل جميع Seeders
```bash
php artisan db:seed
```

### تشغيل Seeder محدد
```bash
php artisan db:seed --class=PositionSeeder
php artisan db:seed --class=ApplicantSeeder
```

### إعادة تعيين قاعدة البيانات مع Seeders
```bash
php artisan migrate:fresh --seed
```

## البيانات المنشأة

### الوظائف
- **5 وظائف** نشطة
- كل وظيفة تحتوي على:
  - اسم الوظيفة
  - وصف مفصل
  - شروط واضحة

### المتقدمين
- **11 متقدم** موزعين على الوظائف:
  - 3 متقدمين لمنصب مدير عام
  - 2 متقدمين لكل وظيفة أخرى
- كل متقدم يحتوي على:
  - بيانات شخصية كاملة
  - مؤهل علمي
  - خبرات سابقة مفصلة
  - مهارات متعددة
  - نقاط تلقائية محسوبة (0-100)
  - تقييم (4-5 نجوم)

## التحقق من البيانات

بعد تشغيل Seeders، يمكنك التحقق من البيانات:

```bash
php artisan tinker
```

ثم:
```php
// عدد الوظائف
App\Models\Position::count();

// عدد المتقدمين
App\Models\Applicant::count();

// المتقدمين مع النقاط
App\Models\Applicant::orderBy('score', 'desc')->get();

// توزيع المتقدمين حسب الوظيفة
App\Models\Position::withCount('applicants')->get();
```

## ملاحظات

- يتم حساب النقاط التلقائية تلقائياً عند إنشاء المتقدم
- جميع المتقدمين في حالة "قيد المراجعة" (under_review)
- البيانات باللغة العربية
- البريد الإلكتروني وهمي (example.com)
