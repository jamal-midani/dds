# التدفقات (Flows)

## تدفق تقديم طلب توظيف

1. المستخدم يزور الموقع
2. يعرض قائمة الوظائف المتاحة
3. يختار وظيفة وينقر على "تقديم الآن"
4. يملأ نموذج التقديم:
   - البيانات الشخصية
   - المؤهل العلمي
   - الخبرات
   - المهارات
   - رفع السيرة الذاتية
5. إرسال الطلب
6. النظام:
   - يحفظ البيانات
   - يحسب النقاط التلقائية
   - يولد ملخص Gemini AI (خلفياً)
7. عرض رسالة نجاح

## تدفق إدارة الوظائف (Admin)

1. تسجيل الدخول إلى لوحة التحكم
2. الانتقال إلى "الوظائف"
3. إضافة/تعديل/حذف وظيفة
4. تفعيل/إلغاء تفعيل وظيفة

## تدفق إدارة المتقدمين (Admin)

1. تسجيل الدخول إلى لوحة التحكم
2. الانتقال إلى "المتقدمين"
3. عرض قائمة المتقدمين
4. استخدام الفلاتر للبحث:
   - حسب المنصب
   - حسب الحالة
   - حسب النقاط
5. عرض تفاصيل المتقدم
6. تقييم المتقدم (1-5 نجوم)
7. تغيير حالة الطلب

## تدفق دعم اتخاذ القرار

1. تسجيل الدخول إلى لوحة التحكم
2. الانتقال إلى "دعم اتخاذ القرار"
3. اختيار المنصب
4. اختيار معيار الترتيب:
   - النقاط
   - الخبرة
   - المؤهل العلمي
   - العمر
   - التقييم
5. عرض قائمة المتقدمين مرتبة
6. عرض أفضل 3 متقدمين (Gemini AI)
7. اتخاذ القرار النهائي
