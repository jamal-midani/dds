# واجهات النظام

## الواجهة العامة (Public Interface)

### صفحة الوظائف المتاحة
- **المسار**: `/` أو `/positions`
- **الوصف**: عرض قائمة الوظائف المتاحة
- **المكونات**:
  - بطاقات الوظائف
  - زر "تقديم الآن" لكل وظيفة

### نموذج التقديم
- **المسار**: `/positions/{id}/apply`
- **الوصف**: نموذج تقديم طلب توظيف
- **الحقول**:
  - الاسم الكامل
  - البريد الإلكتروني
  - رقم الهاتف
  - العمر
  - المؤهل العلمي
  - الخبرات السابقة
  - المهارات
  - رفع السيرة الذاتية (PDF)

### صفحة النجاح
- **المسار**: `/application/success`
- **الوصف**: رسالة تأكيد بعد إرسال الطلب

## لوحة التحكم (Admin Panel)

### إدارة الوظائف
- **المسار**: `/admin/positions`
- **الوظائف**:
  - عرض قائمة الوظائف
  - إضافة وظيفة جديدة
  - تعديل وظيفة
  - حذف وظيفة
  - تفعيل/إلغاء تفعيل

### إدارة المتقدمين
- **المسار**: `/admin/applicants`
- **الوظائف**:
  - عرض قائمة المتقدمين
  - البحث والتصفية
  - عرض تفاصيل المتقدم
  - تحميل السيرة الذاتية
  - تقييم المتقدم
  - تغيير حالة الطلب

### دعم اتخاذ القرار
- **المسار**: `/admin/applicants/decision-support`
- **الوظائف**:
  - اختيار المنصب
  - ترتيب المتقدمين
  - عرض أفضل 3 متقدمين (Gemini AI)
  - عرض جميع المتقدمين
