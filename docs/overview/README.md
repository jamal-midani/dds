# نظرة عامة على المشروع

## الهدف

يهدف المشروع إلى تطوير نظام معلومات إداري يساعد الجهة المختصة (الوزير) على اتخاذ القرار الأنسب لاختيار مدير عام لشركة من بين عدة مرشحين.

## الأهداف الرئيسية

1. توفير واجهة بسيطة للمتقدمين لتقديم طلبات التوظيف
2. إدارة شاملة للوظائف والمتقدمين من خلال لوحة تحكم
3. دعم اتخاذ القرار من خلال:
   - نظام نقاط تلقائي
   - تحليل ذكي باستخدام Gemini AI
   - ترتيب وفلترة المتقدمين

## التقنيات المستخدمة

- **Backend**: Laravel 12
- **Frontend**: Blade Templates + TailwindCSS
- **Admin Panel**: Filament 3
- **Database**: MySQL
- **AI Integration**: Google Gemini API

## المميزات الرئيسية

### للمستخدمين
- عرض الوظائف المتاحة
- تقديم طلبات التوظيف بسهولة
- رفع السيرة الذاتية

### للمسؤولين
- إدارة كاملة للوظائف والمتقدمين
- نظام تقييم تلقائي
- تحليل ذكي باستخدام AI
- دعم اتخاذ القرار
